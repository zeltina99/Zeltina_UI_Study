Detailed Description

무기 전용 수치 데이터 구조체

FItemBaseStats를 상속받아 공격력, 사거리 등 전투에 필요한 구체적인 스탯을 정의합니다.

Constructor & Destructor Documentation

◆ FWeaponStats()

FWeaponStats::FWeaponStats ( )
inline

무기 스탯 생성자

생성 시 ItemType을 자동으로 'Weapon'으로 초기화하여, 데이터 테이블 작성 시 타입을 일일이 지정하는 번거로움을 줄입니다.

◆ AttackPower

float FWeaponStats::AttackPower

기본 공격력 (Base Damage)

무기의 순수 데미지 값입니다. GameplayEffectExecutionCalculation(ExecCalc)에서 캐릭터의 스탯(Strength)과 합산되어 최종 데미지가 계산됩니다.

◆ AttackSpeed

float FWeaponStats::AttackSpeed

공격 속도 배율 (Attack Speed Multiplier)

기본값 1.0을 기준으로, 높을수록 공격 속도가 빨라집니다. 예: 1.2 = 20% 더 빠름. 애니메이션 재생 속도(PlayRate)에 곱해지거나 쿨타임 감소에 영향을 줍니다.

float FWeaponStats::CritDamage

치명타 발생 시 기본 데미지에 곱해지는 배율입니다. 예: 1.5 = 150% 데미지 (기본 100 + 추가 50). 2.0 = 200% 데미지.
