Detailed Description

스테이지 정보를 정의하는 데이터 테이블 행(Row) 구조체입니다.

기획자가 엑셀(csv) 또는 에디터의 테이블에서 각 스테이지의 정보를 설정합니다.

◆ StageIndex

int32 FStageData::StageIndex

스테이지 순서 및 해금 조건을 판단하는 인덱스입니다.

1-1 스테이지는 인덱스 1, 1-2 스테이지는 인덱스 2, ... 1-5는 5이며, 유저가 1을 클리어하면 Index가 2인 스테이지가 해금되는 방식이다.
