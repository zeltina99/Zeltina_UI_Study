Detailed Description

인벤토리 그리드 내의 단일 아이템/캐릭터 슬롯 UI입니다.

  • 캐릭터 또는 아이템 데이터를 받아 아이콘, 테두리 색상, 레벨 등을 표시합니다.
  • 보유 여부(Owned)에 따라 잠금(Lock) 아이콘을 표시하거나 클릭 이벤트를 제어합니다.
