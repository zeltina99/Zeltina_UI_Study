Detailed Description

소환 화면의 좌측 탭(카테고리 선택)을 전담하는 위젯.

  • 버튼 클릭 입력을 받아 현재 탭 상태를 변경한다.
  • 변경된 상태를 델리게이트(OnTabChanged)를 통해 상위 위젯에 알린다.
  • 선택된 탭은 비활성화(Disabled) 처리하여 시각적 피드백을 제공한다.
