Detailed Description

게임의 전역 데이터 및 레벨 이동 간의 비동기 로딩을 관리하는 클래스입니다.

StreamableManager를 사용하여 에셋을 미리 메모리에 로드한 후 OpenLevel을 호출합니다.

◆ MaxClearedStageIndex

int32 UMyGameInstance::MaxClearedStageIndex = 0

현재까지 클리어한 최고 스테이지 인덱스입니다.

0이면 "아무것도 안 깸(1-1 도전 중)", 1이면 "1-1 깸(1-2 도전 가능)"

Note
나중에는 SaveGame에서 불러와야 합니다.
