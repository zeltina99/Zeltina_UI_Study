Detailed Description

가챠 배너 이미지와 천장(Pity) 정보를 표시하는 위젯. [SRP 준수]

  • 상위 위젯으로부터 현재 가챠 타입(Character/Weapon)을 받아 배너 이미지와 타이틀을 갱신한다.
  • 천장 카운트 데이터를 받아 텍스트를 갱신한다.
  • 철저하게 '보여주는 역할(View)'만 수행하며 로직을 포함하지 않는다.
