Detailed Description

인게임 HUD에서 개별 스킬 슬롯의 UI 표시 및 상호작용을 담당하는 위젯.

  • SRP: 스킬 데이터(아이콘) 표시와 쿨타임(Cooldown) 시각화 로직만 담당합니다.
  • Data-Driven: InitializeSkill 함수를 통해 외부 데이터를 주입받아 초기화합니다.
