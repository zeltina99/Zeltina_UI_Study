Detailed Description

상세 정보 패널을 담당하는 위젯 클래스.

  • 캡슐화: UI 컴포넌트는 외부에서 접근 불가능하며, 오직 UpdateInfo 함수로만 제어됩니다.
  • 최적화: 불필요한 렌더링 갱신을 막기 위해 데이터가 유효할 때만 UI를 갱신합니다.
