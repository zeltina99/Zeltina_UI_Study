Detailed Description

타일 뷰(Tile View) 리스트 항목에 데이터를 전달하기 위한 래퍼(Wrapper) 클래스.

UMG Tile View나 List View는 UObject 타입의 객체만 관리할 수 있으므로, 구조체(struct) 형태의 데이터를 이 클래스에 담아서 UI 위젯에 전달하는 역할을 한다. 하나의 클래스로 캐릭터와 무기 데이터를 모두 처리하기 위해 공용으로 사용되며, bIsCharacter 플래그를 통해 현재 데이터의 타입을 구분한다.
