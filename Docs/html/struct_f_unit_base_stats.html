Detailed Description

모든 유닛(플레이어, 몬스터, 소환수)의 공통 기초 능력치를 정의하는 부모 구조체

GAS AttributeSet의 초기화 데이터로 사용됩니다. 'Base' 접두사가 붙은 수치는 장비나 버프가 적용되지 않은 순수 스탯을 의미합니다.
