Detailed Description

AI가 제어하는 유닛(몬스터, 패밀리어)이 추가로 가지는 전투 정보 구조체입니다.

플레이어와 달리 무기(Weapon)에 의존하지 않고 자체적인 공격 주기와 사거리를 가집니다.

Member Data Documentation

◆ AttackInterval

float FAIUnitStats::AttackInterval

공격 속도 / 쿨타임 (Attack Speed)

초 단위(Seconds)입니다. 이 시간마다 한 번씩 공격을 시도합니다.

Note
값이 낮을수록 공격 속도가 빠릅니다. (예: 1.5 = 1.5초마다 1회 공격)

◆ AttackRange

float FAIUnitStats::AttackRange

사거리 (Attack Range)

이 거리 안에 타겟이 들어오면 이동을 멈추고 공격을 시도합니다.

Note
근거리는 보통 100~150, 원거리는 600~1000 정도를 사용합니다.
