Detailed Description

가챠 소환 버튼(1회/10회) 및 비용 표시를 전담하는 위젯. [SRP 준수]

  • 버튼 클릭 시 델리게이트(OnSummonRequest)를 통해 상위 위젯에 소환 횟수를 전달한다.
  • 상위 위젯의 요청에 따라 소모 비용 텍스트를 갱신한다(Data-Driven View).
