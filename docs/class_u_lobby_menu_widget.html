Detailed Description

로비 우측의 메인 메뉴 버튼들(스테이지, 상점, 캐릭터 등)을 관리하는 위젯입니다.

Note
버튼 클릭 시 직접 로직을 수행하지 않고, OnMenuSelected 델리게이트를 통해 상위 위젯으로 이벤트를 전파합니다.
