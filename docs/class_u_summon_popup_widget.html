Detailed Description

가챠 시스템의 최상위 컨테이너 위젯 (Mediator Pattern 적용).

  • 하위 부품 위젯(Tab, Action, Banner)을 소유하고 조율한다.
  • 각 부품에서 발생한 이벤트를 감지하여 로직 컴포넌트(GachaComponent)에 전달한다.
  • 컴포넌트의 데이터 변경 사항을 각 부품에 전파하여 UI를 갱신시킨다.
