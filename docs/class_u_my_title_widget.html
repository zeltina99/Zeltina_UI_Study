Detailed Description

게임의 타이틀 화면을 관리하는 위젯 클래스입니다.

Note
화면 전체를 덮는 투명 버튼을 통해 'Touch to Start'기능을 구현하며, 텍스트 깜빡임 애니메이션을 재생합니다.
